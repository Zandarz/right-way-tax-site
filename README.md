# Right Way Tax Consulting & More - website static

Site de prezentare pentru firmă de contabilitate, fiscalitate, HR, ONRC și expertize extrajudiciare.

## Fișiere
- `index.html` - structura site-ului
- `styles.css` - design responsive
- `script.js` - meniul mobil și anul automat în footer
- `assets/` - logo firmă și siglă CECCAR

## Publicare pe GitHub Pages

1. Creează un cont pe GitHub sau intră în contul existent.
2. Creează un repository nou, de exemplu: `right-way-tax-site`.
3. Încarcă toate fișierele din acest folder în repository.
4. Intră în repository la `Settings` → `Pages`.
5. La `Build and deployment`, alege:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Salvează. GitHub va genera un link de forma:
   `https://username.github.io/right-way-tax-site/`

## Formular de contact

În `index.html`, caută:
`https://formspree.io/f/REPLACE_WITH_YOUR_FORMSPREE_ID`

Creează un formular gratuit pe Formspree, apoi înlocuiește textul `REPLACE_WITH_YOUR_FORMSPREE_ID` cu ID-ul primit.

## Recomandare domeniu

După publicare, cumpără un domeniu, de exemplu:
- `rightwaytax.ro`
- `rightwaytaxconsulting.ro`
- `rightwayconta.ro`

Domeniul se poate conecta la GitHub Pages prin DNS.
